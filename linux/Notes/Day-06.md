##  What is a Linux user?
A user is one of persons who own an account on a system 
## Why do groups exist?
To categorize people that have certain permissions into sections 
## Why doesn't Linux let everyone use sudo?
If everyone is allowed to make a root command using sudo , there will be unathorized usage .
## What is the difference between a normal user and the root user?
A normal user is everyone else - limited to non admin commands , while a root user who is allowed the use of all admin commands   .     
## What command tells you who you're currently logged in as?
whoami
